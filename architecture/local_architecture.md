1. Installed Docker
        │
        ▼
Now your laptop can run containers.

        │
        ▼
2. Installed kubectl
        │
        ▼
Now your laptop has a tool to communicate with Kubernetes.

        │
        ▼
3. Installed Minikube
        │
        ▼
Now your laptop can create a local Kubernetes cluster.

        │
        ▼
4. Ran:
   minikube start --driver=docker
        │
        ▼
Docker created a Minikube container.
        │
        ▼
Kubernetes was installed inside that container.
        │
        ▼
kubectl was configured to connect to that cluster.
        │
        ▼
✅ You now have a working Kubernetes cluster.
